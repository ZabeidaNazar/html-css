Css

Css стилі записуються в атрибуті style:
стиль: значення; стиль: значення

Наприклад задамо параграфу з текстом 'Hello world!' зелений фон і жовтий колір тексту
Код виглядатиме ось так:
<p style="background-color: green; color: yellow">Hello world!</p>
Результат:

Hello world!

Css стилі

Css стилі Призначення Приклади значень
border Межа
(товщина тип колір)		 1px solid black
3px solid green
background-color Колір фону red, green, blue...
#f4ffe5, #FFFF00
